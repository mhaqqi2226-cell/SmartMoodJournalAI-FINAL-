UNTUK YTH DOSEN PENGUJI
==============================================

Jika Bapak/Ibu melihat file ini, berarti source code ini baru saja di-download dari GitHub.

Karena batasan file upload, folder sistem 'node_modules' tidak disertakan (ini standar pemrograman Node.js).
Namun jangan khawatir, saya sudah membuat script otomatis.

CARA MENJALANKAN (SANGAT MUDAH):
1. Pastikan laptop terkoneksi INTERNET (untuk download library di awal saja).
2. Double klik file bernama "start.bat".
3. Script akan otomatis mendeteksi bahwa ini pertama kali dijalankan, lalu akan melakukan instalasi sistem (npm install) secara otomatis.
4. Tunggu sampai selesai, lalu menu aplikasi akan muncul.
5. Pilih [1] untuk membuka Aplikasi Desktop.

TROUBLESHOOTING:
- Jika muncul error "node not recognized", mohon pastikan aplikasi Node.js sudah terinstall di laptop Bapak/Ibu.
